get

prime?
(prime? n)
Checks whether an integer (long or lower) is a prime. Returns true if yes,
false otherwise.
primes
(primes)
Returns a lazy sequence of all primes in sorted order.

take
(take n)
Returns the n first primes as an unmodifiable list.

take-below
(take-below n)
